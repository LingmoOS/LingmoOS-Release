# SDDM theme of CuteOS

![screenshot](screenshot.png)

## Dependent package

For Debian/Ubuntu:
```shell
apt install build-essential cmake extra-cmake-modules qtbase5-dev qtdeclarative5-dev qtquickcontrols2-5-dev qttools5-dev qttools5-dev-tools
```

## Build
```shell
mkdir build
cd build
cmake ..
make
```
